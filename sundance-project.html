Sundance College Website Revamp

Sundance College Website Revamp and Migration Project

I successfully led the end-to-end migration and redevelopment of the Sundance College website, transforming it into a fully functional, scalable, and user-friendly platform to support the institution's growing needs.

Key Responsibilities and Achievements:

  • Infrastructure & Hosting: Migrated the existing website from Laravel framework to WordPress on Amazon Web Services (AWS) LightSail, ensuring optimized performance, security, and cost-effective hosting.
  • Database Management: Handled complete MySQL database management, including data transfer, optimization, and backup strategies for seamless operations.
  • Content Management & Customization: Collaborated closely with the marketing team to manage and update content using WordPress CMS, enhancing website usability and SEO performance.
  • eCommerce Development: Built and implemented a robust WooCommerce-based eCommerce platform to sell online courses directly from the website, enabling seamless transactions and customer experiences.
  • LMS Integration: Integrated Moodle LMS with the website to automate student enrollment and course access upon successful purchases.
  • Technologies Used: Utilized HTML, XML, PHP, CSS, JavaScript for custom development and tailoring website functionalities to meet organizational needs.
  • Monitoring & Analytics: Set up Google Analytics 4 (GA4) for data tracking, performance monitoring, and providing actionable insights for marketing and management teams.
  • Backup & Security: Designed and implemented regular backup solutions and security measures to protect the website from vulnerabilities and ensure high availability.
  • Custom Implementations: Assisted in other custom API integrations and third-party services to extend the website's capabilities.
  • Customer Support: Acted as a point of contact for customer queries, addressing technical issues and ensuring a smooth user experience.
  • Internal Documentation & Training: Authored technical documentation and guides about the website's functionality and processes to support internal teams, including non-technical members.

Outcome & Impact:

  • Achieved a 25% increase in website traffic and course conversions post-launch.
  • Improved website speed, security, and uptime, contributing to higher user satisfaction.
  • Streamlined course sales and enrollment processes through integrated eCommerce and LMS solutions.
